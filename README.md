# 🤖 AI-Post-Generator: أداة توليد المنشورات الذكية للواتساب والتليجرام

![AI Post Generator Banner](https://via.placeholder.com/1200x400?text=AI+Post+Generator+Banner)

## 🌟 نظرة عامة على المشروع

**AI-Post-Generator** هي أداة قوية تعتمد على الذكاء الاصطناعي (AI) لتوليد منشورات جذابة واحترافية مخصصة لقنوات التواصل الاجتماعي مثل واتساب وتليجرام. تم تصميم هذه الأداة لمساعدة منسقي الإعلام والمطورين التقنيين على إنشاء محتوى عالي الجودة بسرعة وكفاءة، مما يوفر الوقت والجهد في عمليات التسويق والتواصل.

بصفتي **مطور تقني ومنسق إعلامي (Technical Developer + Media Coordinator)**، أدرك أهمية المحتوى الجذاب والموجه. يهدف هذا المشروع إلى سد الفجوة بين التقنية والإبداع، مما يتيح للمستخدمين توليد منشورات تتناسب مع أهدافهم التسويقية أو الإعلامية، مع الحفاظ على نبرة صوت موحدة وأسلوب احترافي.

## ✨ الميزات الرئيسية

-   **توليد منشورات مدعومة بالذكاء الاصطناعي**: استخدام نماذج لغوية متقدمة (مثل OpenAI GPT) لإنشاء نصوص فريدة ومبتكرة.
-   **قوالب قابلة للتخصيص**: دعم قوالب JSON لتحديد أنواع المنشورات المختلفة (إعلانات، نصائح، تحفيز، أسئلة تفاعلية) مع أمثلة إرشادية.
-   **دعم اللغة العربية**: توليد منشورات باللغة العربية الفصحى مع استخدام الرموز التعبيرية المناسبة.
-   **سهولة الاستخدام**: واجهة سطر أوامر بسيطة لتحديد الموضوع والقالب المطلوب.
-   **تخزين الأمثلة**: حفظ المنشورات المولدة تلقائياً في مجلد `examples/` للرجوع إليها مستقبلاً.
-   **مرونة التكامل**: يمكن تكييفها بسهولة للتكامل مع أنظمة إدارة المحتوى أو أدوات النشر الآلي.

## 🛠️ التقنيات المستخدمة

-   **Python 3.x**: لغة البرمجة الرئيسية للمشروع.
-   **OpenAI API**: لتوليد النصوص باستخدام نماذج الذكاء الاصطناعي (مثل `gpt-4.1-mini`).
-   **`python-dotenv`**: لإدارة متغيرات البيئة (مثل مفتاح API) بشكل آمن.
-   **JSON**: لتخزين قوالب المنشورات بشكل منظم.

## 💡 المهارات المكتسبة من هذا المشروع

من خلال تطوير هذا المشروع، تم تعزيز المهارات التالية:

1.  **البرمجة باستخدام Python**: تطوير تطبيقات سطر الأوامر وهيكلة الكود.
2.  **التعامل مع واجهات برمجة تطبيقات الذكاء الاصطناعي (AI APIs)**: فهم وتطبيق OpenAI API لتوليد المحتوى.
3.  **إدارة البيانات المنظمة**: استخدام JSON لتخزين وإدارة القوالب.
4.  **إدارة متغيرات البيئة**: تأمين المعلومات الحساسة باستخدام ملفات `.env`.
5.  **التفكير التصميمي للمحتوى**: بناء قوالب فعالة لتوجيه الذكاء الاصطناعي نحو توليد أنواع محددة من المنشورات.
6.  **أتمتة المحتوى**: فهم كيفية أتمتة عملية إنشاء المحتوى لوسائل التواصل الاجتماعي.
7.  **التعليق والتوثيق**: كتابة كود نظيف وموثق بشكل جيد.

## 🚀 كيفية التشغيل

### 1. المتطلبات الأساسية

تأكد من تثبيت Python 3.x على نظامك.

### 2. استنساخ المستودع

```bash
git clone https://github.com/waleedshamhan2050-sys/AI-Post-Generator.git
cd AI-Post-Generator
```

### 3. إعداد البيئة الافتراضية (اختياري وموصى به)

```bash
python -m venv venv
source venv/bin/activate  # لنظامي Linux/macOS
venv\Scripts\activate    # لنظام Windows
```

### 4. تثبيت المكتبات المطلوبة

```bash
pip install -r requirements.txt
```

### 5. إعداد مفتاح OpenAI API

1.  احصل على مفتاح API الخاص بك من [منصة OpenAI](https://platform.openai.com/api-keys).
2.  قم بإنشاء ملف جديد باسم `.env` في المجلد الرئيسي للمشروع (بجانب `main.py`).
3.  أضف مفتاح API الخاص بك إلى الملف بالشكل التالي:
    ```
    OPENAI_API_KEY="your_openai_api_key_here"
    ```

### 6. تشغيل البرنامج

```bash
python main.py
```

سيطلب منك البرنامج إدخال موضوع المنشور واختيار القالب من القائمة المتاحة.

## 📝 تخصيص القوالب والأسلوب

يمكنك تخصيص سلوك الأداة بشكل كبير من خلال تعديل ملف `templates.json`:

-   **إضافة قوالب جديدة**: قم بإضافة كائنات JSON جديدة إلى الملف، مع تحديد `description` (وصف القالب) و `example` (مثال توضيحي للمنشور المطلوب).
-   **تعديل القوالب الحالية**: يمكنك تغيير الوصف والأمثلة للقوالب الموجودة لتناسب احتياجاتك.
-   **تغيير أسلوب التوليد**: يمكن تعديل `prompt` في دالة `generate_post` داخل `main.py` لتوجيه الذكاء الاصطناعي نحو أسلوب كتابة معين (رسمي، فكاهي، تقني، إلخ) أو إضافة قيود أخرى (مثل عدد الكلمات، استخدام هاشتاجات محددة).
